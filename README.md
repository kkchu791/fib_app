# Fib App

Get the next or previous number of your current fibanacci number.


------------------

Live site can be found here: [fib_client](https://fib-client.surge.sh) 


## Setup backend instructions

```
$ cd fib_api
$ bundle install
$ rake db:migrate
$ rails s -p 3001

```


## Setup frontend instructions

```
$ cd fib_client
$ yarn install
$ yarn start

```
